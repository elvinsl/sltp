
# SLTP (StopLoss & TargetProfit)

Calculate stop loss and target profit with web interface.



## Installation

Install sltp with git

```bash
  git clone "https://github.com/elvinsl/sltp.git"
  cd sltp
```
    
## Usage/Examples

### Windows
```
python app.py
```

### Linux
```
python3 app.py
```
